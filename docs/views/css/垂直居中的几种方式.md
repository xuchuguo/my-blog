1. 设置padding上下相等  // 前提父元素的height没写
    ``` css
    <div class="ts">
        <p>这里是第一句</p>
        <p>这里是第二句</p>
    </div>

    .ct{
        padding: 10px 0;
        text-align: center;
        background: #eee;
    }
    ```

2. table标签自带功能
    ``` html
    <table class = "content">
        <tr>
            <td>
                这是一串文字
            </td>
        </tr>
    </table>
    ```

3. 绝对定位
    ``` css
    <div id = "content">
    <div id = "main">
        这里是第三句
    </div>
    </div>

    #content {
        position:relative;
        width: 100vw;
        height: 100vh;  /* 设置宽高以显示居中效果*/
    }
    #main {
        position:absolute;
        top:50%;
        left:50%;
        width: 200px;
        height: 200px;

        margin-top:-100px; /*元素高度的一半*/
        margin-left:-100px; /*元素宽度的一半*/

        transform:translate(-50%,-50%); /*容器没有设置宽高才用这个*/
    }
    ```

3. flex
    ``` css
    <div id = "content">
    <div id = "main">
        这是第四局话
    </div>
    </div>

    #content {
        width: 100vw;
        height: 100vh;  /* 设置宽高以显示居中效果*/
        display: flex;  /* 弹性布局 */
        align-items: center;  /* 垂直居中 */
        justify-content: center;  /* 水平居中 */
    }
    ```
